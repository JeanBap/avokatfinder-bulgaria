Legal Term

What is Ипотека (Mortgage)?

A legal charge on real property that secures a loan in Bulgaria.

Key Facts — Ипотека (Mortgage)
  • Must be registered in the Property Register
  • Provides security for loans using property as collateral
  • Multiple mortgages allowed with priority by registration date
  • Foreclosure available if borrower defaults
  • Registration creates legal notice of creditor interest

An Ипотека (Ipoteka) is a fundamental financial instrument in Bulgarian property law that allows property owners to borrow money using real estate as collateral. It is governed by the Bulgarian Law on Property Register. The system provides security for both lenders and borrowers.

Registration of an Ипотека with the Property Register (Регистър на имотите) is essential for its validity and enforceability. The Property Register maintains public records of all property rights and encumbrances. Priority among mortgages is determined by registration date.

Bulgarian law permits multiple Ипотеки on the same property, each with priority determined by registration order. First mortgages have priority in case of foreclosure, followed by subordinate mortgages. This system allows borrowers flexibility in obtaining financing.

In case of borrower default, the mortgage holder can initiate foreclosure proceedings under Bulgarian law. These procedures are regulated to protect both creditor and debtor interests. The process typically involves formal notice and opportunity for the debtor to cure the default.

The Bulgarian mortgage system uses standardized appraisal procedures to determine property values and acceptable loan amounts. Lenders typically limit mortgages to a percentage of the property's value to ensure adequate security. This approach maintains stability in the property market.

Common Mistake: Believing that an agreement to mortgage property is valid without Property Register registration. Registration is legally essential.
Expert Tip: Before purchasing property in Bulgaria, obtain a certified extract from the Property Register showing all mortgages and encumbrances. This reveals financial obligations.

Frequently Asked Questions

Can a property have multiple Ипотеки in Bulgaria?

Yes, multiple mortgages are allowed with priority determined by registration date.

What is the typical mortgage term in Bulgaria?

10-25 year mortgages are common, though terms vary based on lender policies.

How is the mortgage amount determined in Bulgaria?

Lenders typically limit mortgages to 50-70% of the property's appraised value.
